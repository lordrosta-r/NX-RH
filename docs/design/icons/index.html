Import

Toujours importer depuis le barrel export icons/index.js : 

import { SunIcon, MoonIcon, GlobeIcon } from '../components/ui/icons'

<SunIcon size={16} />
<MoonIcon size={16} color="var(--th-controls-icon)" />
<GlobeIcon size={14} color="currentColor" strokeWidth={1.5} />

Ajouter une icône

  1. Créer client/src/components/ui/icons/NomIcon.jsx
  2. Respecter les props size, color, strokeWidth
  3. Toujours aria-hidden="true" sur le SVG
  4. Ajouter l'export dans icons/index.js
  5. Documenter ici dans cette page
